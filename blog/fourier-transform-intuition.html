理解傅里叶变换：从直观到数学

2026年5月20日 数学
傅里叶变换信号处理数学分析

引言

傅里叶变换是数学和工程领域最具影响力的工具之一。它告诉我们：任何信号都可以分解为不同频率的正弦波的叠加。

从直观理解开始

想象你在听一首交响乐。你的耳朵同时听到小提琴、大提琴、长笛等乐器的声音，但你仍然能分辨出每种乐器的音色。这是因为你的耳朵天然地进行了"频率分析"——这正是傅里叶变换在做的事情。

连续傅里叶变换

对于一个连续函数 f ( t ) f(t) f(t)，其傅里叶变换定义为：

f ^ ( ξ ) = ∫ − ∞ ∞ f ( t )   e − 2 π i t ξ   d t \hat{f}(\xi) = \int_{-\infty}^{\infty} f(t) \, e^{-2\pi i t \xi} \, dt f^​(ξ)=∫−∞∞​f(t)e−2πitξdt

其逆变换为：

f ( t ) = ∫ − ∞ ∞ f ^ ( ξ )   e 2 π i t ξ   d ξ f(t) = \int_{-\infty}^{\infty} \hat{f}(\xi) \, e^{2\pi i t \xi} \, d\xi f(t)=∫−∞∞​f^​(ξ)e2πitξdξ

直观解释

公式中的 e − 2 π i t ξ e^{-2\pi i t \xi} e−2πitξ 可以理解为"频率为 ξ \xi ξ 的探针"。积分的过程就是在计算信号 f ( t ) f(t) f(t) 与这个探针的相似度。

应用场景

1. 音频处理

MP3 压缩的核心就是傅里叶变换：将音频信号变换到频域，滤除人耳不敏感的频率分量。

2. 图像处理

JPEG 压缩使用离散余弦变换（DCT，傅里叶变换的变体）：

3. 卷积定理

傅里叶变换最重要的性质之一：时域卷积等价于频域乘积。

F { f ∗ g } = F { f } ⋅ F { g } \mathcal{F}\{f * g\} = \mathcal{F}\{f\} \cdot \mathcal{F}\{g\} F{f∗g}=F{f}⋅F{g}

直观可视化

想象一个信号 f ( t ) = sin ⁡ ( 2 π ⋅ 3 t ) + 0.5 sin ⁡ ( 2 π ⋅ 10 t ) f(t) = \sin(2\pi \cdot 3t) + 0.5\sin(2\pi \cdot 10t) f(t)=sin(2π⋅3t)+0.5sin(2π⋅10t)，它是 3Hz 和 10Hz 正弦波的叠加。傅里叶变换会在这两个频率处出现峰值：

在 ξ = 3 \xi = 3 ξ=3 处有一个高峰，在 ξ = 10 \xi = 10 ξ=10 处有一个稍矮的高峰，其他频率处接近零。

总结

傅里叶变换的核心思想可以用一句话概括：

将信号从时域映射到频域，让我们看到"声音的颜色"。
